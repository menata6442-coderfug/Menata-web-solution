# Menata Web Solutions

A professional website design and development portfolio/service website built with HTML, CSS, and vanilla JavaScript.

## Features

- **Responsive Design**: Fully responsive layout that works on mobile, tablet, and desktop
- **Modern UI**: Professional design with smooth animations and transitions
- **Mobile Menu**: Hamburger menu for mobile navigation
- **Contact Form**: Contact form that emails submissions to `menata6442@gmail.com` with a sending state and redirect
- **Service Showcase**: Display of various web development services
- **SEO Optimized**: Includes meta tags for search engine optimization

## Project Structure

Menata-web-solution/
├── index.html # Main HTML file
├── styles.css # CSS styling
├── script.js # JavaScript functionality
└── README.md # This file

## Features Overview

### Navigation
- Sticky header with logo and navigation menu
- Mobile hamburger menu
- Smooth scrolling to sections

### Hero Section
- Eye-catching banner with call-to-action buttons
- Professional messaging

### Services
- Grid layout showcasing 6 core services:
  - Website Design
  - Website Development
  - E-commerce Solutions
  - Business Websites
  - Website Maintenance
  - SEO Optimization

### Contact Section
- Direct contact methods (phone and email)
- Contact form that sends via FormSubmit to `menata6442@gmail.com`
- Sending state that disables the button and shows progress
- Redirect to a “Message Sent” section after submission

### JavaScript Functionality
- Hamburger menu toggle
- Sending state for the contact form
- Responsive interactions

## Getting Started

1. Clone or download the project
2. Open `index.html` in your web browser
3. No build process required - it's ready to use!

## Customization

    --gold-color: #ffc107;
    --dark-color: #1a1a1a;
    --light-color: #f5f5f5;
    --text-color: #333;
}
